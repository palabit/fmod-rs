Build helpers for [fmod-core-sys], [fmod-fsbank-sys], and [fmod-studio-sys].
Provides the FMOD lookup and mapping from Rust targets to FMOD targets.

Not intended for use outside those crates. Use them or [fmod-rs] instead.
